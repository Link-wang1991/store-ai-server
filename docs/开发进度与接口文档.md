## 二、已完成接口清单

所有接口统一使用 **GET / POST**，无 PUT / DELETE。

### 2.1 认证 (auth)

| 方法 | 路径 | 说明 |
|------|------|------|
| `POST` | `/api/auth/login` | 邮箱密码登录，返回 JWT Token |

**请求**：
```json
{ "email": "owner@demo.com", "password": "demo123456" }
```

**响应**：
```json
{
  "code": 200,
  "data": {
    "token": "eyJ...",
    "userId": "xxx",
    "employeeId": "xxx",
    "storeId": "xxx",
    "role": "owner",
    "roleLabel": "owner"
  }
}
```

**鉴权**：后续所有请求 Header 带 `Authorization: Bearer {token}`

---

### 2.2 知识库 (knowledge)

| 方法 | 路径 | 说明 | 权限 |
|------|------|------|------|
| `GET` | `/api/knowledge` | 启用的文档列表，支持 `?category=` 筛选 | 全员 |
| `GET` | `/api/knowledge/search?q=&topN=5` | bigram 关键词检索片段 | 全员 |
| `POST` | `/api/knowledge/upload` | 上传文件 + 自动解析切分 | 管理员 |
| `POST` | `/api/knowledge/manual` | 手动录入纯文本 | 管理员 |
| `POST` | `/api/knowledge/{id}/toggle` | 启用/停用 | 管理员 |
| `POST` | `/api/knowledge/{id}/delete` | 删除文档及片段 | 管理员 |

**上传接口** (multipart/form-data)：
```
file:          (文件)
title:         补水修复项目价格表
category:      project
visibleRoles:  owner,manager,consultant
tags:          补水,价格
```

**手动录入**：
```json
{
  "title": "客户嫌贵标准话术",
  "category": "script",
  "content": "一、先认同客户感受...",
  "visibleRoles": ["consultant", "manager"]
}
```

**检索响应**：
```json
{
  "code": 200,
  "data": [
    {
      "id": "xxx",
      "documentId": "xxx",
      "content": "补水修复项目原价...",
      "score": 2.5
    }
  ]
}
```

### 2.3 AI 对话 (chat)

| 方法 | 路径 | 说明 |
|------|------|------|
| `POST` | `/api/chat` | 核心问答接口 |

**请求**：
```json
{
  "question": "补水修复项目多少钱？",
  "sessionId": null,
  "customerId": null
}
```

**回答类型**：
| answerType | 含义 |
|------------|------|
| `knowledge` | 知识库命中，基于门店资料回答 |
| `general` | 无匹配，通用建议 |
| `quick_reply` | 快捷问题模板 |
| `risk` | 高风险（过敏/投诉等），自动拦截升级 |

**当前实现**：评估风险 L1-L4 → bigram 检索 → 结构化模板回答。待对接真实 LLM。
